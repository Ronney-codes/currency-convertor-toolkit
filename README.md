# currency-convertor-toolkit
A SIMPLE C CURRENCY CONVERTOR




## Description
This is a simple C-based currency converter that allows users to convert amounts between a few predefined currencies. The program accepts an amount from the user, provides a menu of conversion options, performs the selected conversion, and displays the final converted value.

## Features
- Accepts user input for the amount to convert
- Provides a menu to select the type of currency conversion
- Converts between predefined currencies (e.g., USD, EUR, KES)
- Displays the converted value clearly

## How to Use
1. Compile the program using your C compiler (e.g., Visual Studio, GCC).
2. Run the executable.
3. Enter the amount you want to convert.
4. Select the desired currency conversion option from the menu.
5. View the converted amount displayed on the screen.

## Requirements
- A C compiler (e.g., GCC, Visual Studio)
- Basic understanding of how to run a C program in your environment

## Example
Enter amount: 100
Select conversion:

USD to KES

KES to USD

USD to EUR
Your choice: 1
Converted amount: 13500 KES
